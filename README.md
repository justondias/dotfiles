# Deps
* nvim
* ripgrep
* nodejs
* python3
* vim-plug
* pynvim
* tmux

# Installation
1. Install pynvim - `python3 -m pip install --user --upgrade pynvim`
2. Install vim-plug to `~/.config/nvim/autoload/plug.vim`
3. Replace `~/.config/nvim/init.vim`
4. Run `:PlusInstall`

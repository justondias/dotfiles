# Deps
* nvim
* nodejs
* ripgrep
* vim-plug - installed to `~/.config/nvim/autoload/plug.vim`
* pynvim - `python3 -m pip install --user --upgrade pynvim`

# Installation
* replace `~/.config/nvim/init.vim`
* Run `:PlusInstall`
